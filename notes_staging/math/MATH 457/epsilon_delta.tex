% title: Epsilon-Delta Continuity
% date: 2026-02-26
% tags: continuity, proofs

\section*{Definition}
A function $f$ is continuous at $a$ if for every $\varepsilon > 0$ there exists $\delta > 0$
such that if $|x-a|<\delta$ then $|f(x)-f(a)|<\varepsilon$.

\section*{Example}
Show $f(x)=3x+1$ is continuous at any $a$.

Let $\varepsilon>0$. Choose $\delta=\varepsilon/3$. If $|x-a|<\delta$, then
\[
|f(x)-f(a)| = |3x+1-(3a+1)| = 3|x-a| < 3\delta = \varepsilon.
\]
